== Description ==

Customize your TopBanner at the top of your website.

TopBanner is a simple plugin that adds a customizable, fixed-position banner at the top of your website. The text of the banner can be easily managed from the WordPress admin panel. Perfect for displaying announcements or important messages.

== Installation ==
1. Upload the `topbanner` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Banner Settings' to configure the banner text.

== Frequently Asked Questions ==
= Can I change the color of the banner? =
Yes, you can modify the CSS in the plugin's stylesheet to change the appearance of the banner.
